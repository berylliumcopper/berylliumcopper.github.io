# Personal GitHub Pages Site

Minimal static website for GitHub Pages with:

- `Home` page for your introduction and project links
- `Gallery` page for pictures
- `Blogs` page for articles

## Structure

- `index.html` - landing page
- `gallery.html` - image gallery
- `blogs/index.html` - blog archive
- `blogs/first-post.html` - sample article
- `assets/css/styles.css` - shared styling
- `assets/images/` - local images

## Customize

1. Replace `Your Name` and the intro text in `index.html`, `gallery.html`, and the blog files.
2. Update the GitHub links on the home page.
3. Replace the placeholder SVG files in `assets/images/` with your own photos.
4. Add a new blog post by copying `blogs/first-post.html` and linking it from `blogs/index.html`.

## Publish With GitHub Pages

1. Create a GitHub repository.
2. Push this folder to the repository.
3. In GitHub, open repository `Settings > Pages`.
4. Set the source to deploy from the main branch root.
5. Save and wait for the site URL to appear.
